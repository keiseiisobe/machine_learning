# Neural Network Classifier

This directory contains a simple neural network classifier for the FashionMNIST dataset.

## `fashion_mnist_classifier.py`

This script defines and trains a neural network using PyTorch to classify images from the FashionMNIST dataset.

### Model Architecture

-   **Input:** 784 pixels (28x28 image)
-   **Hidden Layer 1:** 512 neurons with ReLU activation
-   **Hidden Layer 2:** 512 neurons with ReLU activation
-   **Output Layer:** 10 neurons (for 10 classes)

### Usage

The script can be run directly:
```
python fashion_mnist_classifier.py
```
